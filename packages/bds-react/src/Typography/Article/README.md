# Article
The Article component is used for long form multiline text. This renders a `<p>` element by default.

## Installation

```bash
npm install @lowes-tech/bds-react
```

## Usage

```jsx
import { Article } from '@lowes-tech/bds-react'

<Article>Some Article Text</Article>
```
