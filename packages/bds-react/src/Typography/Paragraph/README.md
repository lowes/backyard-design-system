# Paragraph
The Paragraph component is used for multiline text. This renders a `<p>` element by default.

## Installation

```bash
npm install @lowes-tech/bds-react
```

## Usage

```jsx
import { Paragraph } from '@lowes-tech/bds-react'

<Paragraph>Some Paragraph Text</Paragraph>
```
