# Article
The Caption component is used for annotating. This renders a `<span>` element by default.

## Installation

```bash
npm install @lowes-tech/bds-react
```

## Usage

```jsx
import { Caption } from '@lowes-tech/bds-react'

<Caption>Some Span Text</Caption>
```
